
샤나 인코더를 이용하여 CFR(고정 프레임 레이트) 로 인코딩 해야만, 각 단위 시간 당 동일한 프레임이 할당된다.

프레인 타임라인표(오프닝종료프레임, 아이캐치시작프레임 등을 표시한 표) 는 CFR 이 적용된 비디오 파일을 기준으로 작성되었으므로, 모든 작업을 하기 전에 우선 샤나 인코더 등의 인코더를 이용하여 CFR 을 적용해야한다.

이때, 샤나 인코더에서 화질 손실 없이 인코딩하려면, 샤나 인코더의 인코딩 환경설정에서 영상사이즈 변경을 꺼야함.

비디오 퀄리티는 1에 가까울 수록 원본과 가깝다. 23 정도가 적당한 값이라고 한다 (참고 : https://shana.pe.kr/shanaencoder_tip/100876)


위 설명을 샤나 인코더의 "빠른 설정" 에 설정하면 된다. (첨부 : 블루레이_빠른설정.png)

<"샤나 인코더/빠른 설정/HDR 을 SDR 로 톤 매핑" 설명>

HDR 을 SDR 로 인코딩하면 색감이 떨어지지만, HDR 미지원기기로도 감상할 수 있게된다.
컴퓨터는 HDR 을 항상 지원하는 것 같다.

Shotcut (이하 샷컷) 은 무료 영상 편집 프로그램이다.

- "필터" 기능으로 동영상 크기를 바꾸거나 자막을 넣을 수 있다.

이때, 필터/텍스트 : 단순 으로 텍스트를 넣고, 텍스트를 #frame# 으로 설정하면 현재 프레임이 보인다 (CFR 로 인코딩된 상태이면 프레임 타임라인표와 일치함)
텍스트를 #timecode# 로 설정하면 현재 시간이 보이는데, 굳이 할 필요는 없다. (이미 shotcut  영상 아래에 timecode 가 보이므로)
(참고 : https://forum.shotcut.org/t/how-to-display-frame-number-in-timeline-instead-of-time/31620/)

- 샷컷에서 동영상을 선택하고 화살표로 움직이면 "프레임 단위" 로 이동된다.
